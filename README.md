Data Submission
========

##Shiny application with associated supporting documentation.

* The app checks adult weight status by calculating his/her BMI.

* Input weight in kg and your height in cm into boxes.

* Enter weight and height on the left sidebar, and BMI and your weight status will be show on the right main bar.

* BMI Calculation: Formular 1 : Weight (kg) / [Height (meter)]^2 or Formular 2 : (Weight (lb) / [Height (inch)]^2) * 703

Below is the relationship between BMI and Weight Status

##BMI - Description
* Below 18.5  - Underweight
* 18.5 - 24.9	-Normal
* 25.0 - 29.9	- Overweight
* 30.0 and Above	- Obese
